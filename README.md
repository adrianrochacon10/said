## Ejecutar la Aplicación

1. **Clonar el Repositorio:**
```
git clone https://github.com/adrianrochacon10/said.git
```

2. **Instalar Dependencias:**
```
cd tu-repositorio
npm install
```

3. **Ejecutar en Dispositivo Android/iOS:**
```
npm run android
```
o
```
npm run ios
```
